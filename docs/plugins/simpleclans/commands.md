# 🔧 Comandos do Simple Clans

## Comandos Básicos

| Comando                        | Descrição                              |
|-------------------------------|----------------------------------------|
| `/clan create <tag> <nome>`   | Criar um novo clã                      |
| `/clan invite <jogador>`      | Convidar alguém para o clã             |
| `/clan join <tag>`            | Entrar em um clã                       |
| `/clan list`                  | Ver todos os clãs                      |
| `/clan profile [tag]`         | Ver informações do clã                 |

## Chat

- `/cc <mensagem>` → Chat do clã
- `/ac <mensagem>` → Chat de aliança

## Base e Utilitários

- `/clan home set` → Definir base do clã
- `/clan home` → Teleportar para a base
- `/clan bank` → Gerenciar banco do clã

**Use TAB** para completar os comandos!
