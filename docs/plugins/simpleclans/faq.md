# ❓ Perguntas Frequentes

**Como sair de um clã?**  
`/clan resign`

**Como expulsar alguém?**  
`/clan kick <jogador>` (só leader/trusted)

**Perdi minha base?**  
Use `/clan home`

**Clã foi desfeito?**  
O líder pode recriar ou você pode criar outro.

**Mais dúvidas?** Pergunte no Discord ou no chat do servidor!
